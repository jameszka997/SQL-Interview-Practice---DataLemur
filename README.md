# 🔍 SQL Interview Practice – DataLemur

This repository documents my solutions and workflows for SQL interview practice questions from the [DataLemur](https://datalemur.com/) platform, covering real-world analytical scenarios across a range of industries and difficulty levels.

---

## 🎯 Aim of this Project

The goal of this repository is to practice and deepen my SQL querying skills in an interview-oriented context, working through business-relevant problems that mirror real data analyst and data science interview questions.

---

## 📂 Current Question Sets

### 💊 Pharmacy Analytics (CVS Health)
A series of pharmaceutical sales analysis questions covering profitability ranking, loss identification, and formatted sales reporting by manufacturer.

- **Part 1** – Top 3 most profitable drugs by total profit
- **Part 2** – Manufacturers associated with loss-making drugs and total losses
- **Part 3** – Total drug sales per manufacturer, formatted for dashboard display

---

### 👤 Data Science Candidate Filtering
Identifying candidates proficient in all required skills (Python, Tableau, PostgreSQL) using conditional aggregation.

---

### 🔧 Tesla Assembly Line Analysis
Extracting unfinished parts from a production assembly table using NULL filtering.

---

### 💬 Microsoft Teams Power Users
Identifying the top 2 most active message senders in a given time window using aggregation and ordering.
